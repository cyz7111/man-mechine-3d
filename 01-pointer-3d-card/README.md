# 01 Pointer 3D Card

这是系列里的第一个实验：用鼠标或触控位置控制一个 3D 卡片。

## 你要观察什么

- 输入：指针在屏幕里的 X/Y 位置。
- 映射：X 控制左右旋转，Y 控制上下旋转。
- 反馈：卡片角度、光照位置、状态点和数值面板同时变化。
- 恢复：指针离开区域或点击复位按钮后，系统回到默认状态。

## 运行方式

直接用浏览器打开：

```txt
index.html
```

## 小任务

1. 把最大旋转角度改小，让它更稳。
2. 把最大旋转角度改大，让它更夸张。
3. 试着把 `pointermove` 改成只有按住鼠标时才生效。
4. 记录你更喜欢哪一种，并写下原因。

## 复盘问题

- 你是否能立刻理解自己的操作正在影响什么？
- 数值面板有没有帮助你理解输入和输出的关系？
- 复位按钮是否足够明显？
- 如果这是一个真实产品，这种交互适合用在哪里？
