# Man Machine 3D Interaction Lab

这是一个长期的人机交互练习仓库，目标是把前端能力逐步扩展到空间交互、手势输入、语音控制、传感器输入和 AI 辅助界面。

你不需要一开始就掌握完整的人机交互理论。这个系列会用一个个小而完整的实验，把输入、反馈、状态、空间和可用性慢慢串起来。

## 系列路线

1. `01-pointer-3d-card`
   - 用鼠标或触控位置控制 3D 卡片。
   - 目标：理解输入映射、实时反馈、状态显示和视觉响应。

2. `02-drag-3d-object`
   - 用拖拽控制 3D 物体的位置和旋转。
   - 目标：理解 direct manipulation，也就是直接操作。

3. `03-voice-command-scene`
   - 用语音命令改变场景状态。
   - 目标：理解多模态输入和命令反馈。

4. `04-camera-hand-tracking`
   - 用摄像头和手部识别控制界面。
   - 目标：理解手势输入、识别置信度和误触防护。

5. `05-phone-motion-controller`
   - 用手机陀螺仪控制网页里的 3D 场景。
   - 目标：理解传感器输入和设备协同。

6. `06-ai-agent-control-panel`
   - 做一个人和 AI 共同控制 3D 场景的界面。
   - 目标：理解人机协作、可解释反馈和可撤销操作。

## 学习方式

每个实验都遵循同一个节奏：

1. 先做一个能运行的版本。
2. 写清楚输入是什么、输出是什么、反馈是什么。
3. 加入一个衡量指标，例如响应速度、误触次数、完成任务时间。
4. 记录一次复盘：哪里顺手，哪里别扭，下一版怎么改。
